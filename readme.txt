[url=https://custom.simplemachines.org/mods/index.php?mod=4155][b]SMF-External Image Proxy[/b][/url]

This mod provide SMF 2.1 with the posibility to use external image proxy.

[b][i]Proxy mode[/i][/b]
- Internal mode, which is the smf proxy.php solution
- Easy External mode, which add based on proxy_url + image_url, exp. nginx or https://github.com/willnorris/imageproxy
[b][i]Use case[/i][/b]
You want to use a image proxy for your forum,
but the proxy.php solution is not enough.

This mod run on MySQL, MariaDB and PostgreSQL and
use only hooks.

After install you find the entry under: Configuration > Modification Settings > External Image Proxy
